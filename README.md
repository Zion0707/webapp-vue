# webApp 运行方式

```

1、yarn watch 项目编译监听

2、yarn dev 启动本地端口

3、这里为什么没有使用 devServer 是因为hbuilder修改在模拟器中边改边看中不支持本地端口
 
4、如果有接口请求则使用nginx代理请求，解决跨域问题。

```
